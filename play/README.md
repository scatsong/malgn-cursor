HTML5 비디오 플레이어 샘플

설명
- 이 샘플은 기본 HTML5 비디오 플레이어를 보여줍니다. 재생/일시정지, 탐색바, 볼륨, 전체화면, 자막(트랙 필요), 화질 선택(샘플) 기능을 포함합니다.

사용법
1. 이 저장소의 `index.html`을 브라우저에서 열어 확인합니다. 로컬 파일을 더블클릭하거나 간단한 로컬 서버를 사용하세요.

로컬 서버(권장)
- 파이썬 3가 설치되어 있다면 프로젝트 루트에서 다음 명령을 실행:

  python -m http.server 8000

- 브라우저에서 http://localhost:8000 을 엽니다.

패키지로 실행하기
1) PowerShell 스크립트 사용(Windows)
   프로젝트 루트에서:

   .\start-server.ps1

2) npm 스크립트 사용(선택)
   - 프로젝트에 Node.js가 있다면 다음으로 간단히 시작할 수 있습니다:

     npm install
     npm start

설정 패널 안내
- 플레이어 우측의 ⚙ 버튼을 눌러 자막 관련 설정을 엽니다.
- 프리셋: 현재 스타일을 로컬에 저장하고 불러올 수 있습니다.
- 애니메이션: 전체 속도, 인/아웃 지속시간, 이징을 조절할 수 있습니다.
- 스타일: 폰트, 글자 크기, 색상, 배경 투명도, 하단 여백을 조정하고 저장할 수 있습니다.
- 위치: 왼쪽/중앙/오른쪽 정렬을 선택할 수 있습니다.
 - 프리셋 내보내기/가져오기: 설정 패널에서 프리셋을 JSON 파일로 내보내거나 JSON 파일을 가져와 병합할 수 있습니다.
 - 프리셋 미리보기: 드롭다운에서 프리셋을 선택하면 저장하지 않고도 스타일을 미리 적용해볼 수 있습니다.

추가 기능
- 모두 내보내기: 설정 패널의 "모두 내보내기" 버튼을 눌러 현재 저장된 모든 프리셋을 하나의 JSON 파일로 다운로드합니다.
- 모두 가져오기(병합): JSON 파일을 선택하면 기존 프리셋에 병합됩니다. 동일한 이름의 프리셋은 덮어써집니다.
- 모두 가져오기(대체): 선택한 JSON 파일의 내용으로 로컬에 저장된 모든 프리셋을 대체합니다.
- 초기화(기본값): 로컬에 저장된 모든 프리셋과 사용자 스타일을 삭제하고 페이지를 새로고침하여 기본값으로 되돌립니다.
- 미리보기 자동 롤백: 프리셋을 드롭다운에서 선택하면 미리보기로 스타일이 적용되며, 설정 패널의 "미리보기 자동 롤백"을 켜면 지정된 시간(기본 5초) 후 원래 저장된 스타일로 자동 복원됩니다.

HLS 테스트
- 샘플 HLS 스트림 재생을 위해 플레이어에 "HLS" 버튼이 추가되어 있습니다. 이 버튼을 눌러 HLS 스트림을 켜고 끌 수 있습니다.
- 브라우저가 HLS를 지원하지 않거나 CORS 정책으로 외부 스트림이 차단될 수 있으니, 문제가 발생하면 크롬/파이어폭스 최신 버전에서 테스트하거나 로컬/허용된 HLS 스트림을 사용하세요.

자막 파일
- `subs_ko.vtt` 파일을 추가했습니다. `index.html`에서 자동으로 로드됩니다.
 - 다중 자막 트랙을 추가했습니다. 플레이어의 자막 선택 드롭다운에서 한국어/English/끄기 를 선택할 수 있습니다.

자막 추가
- `video` 요소 안의 `track` 요소의 `src` 속성에 VTT 파일 경로를 넣습니다.

화질 소스 추가
- `source` 요소에 `data-quality` 속성(예: "720p")을 추가하고 `src`를 수정하세요.

저작권
- 샘플 영상은 MDN 예제 비디오입니다.
